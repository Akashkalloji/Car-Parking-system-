Car Parking System 🚗

A simple and user-friendly Car Parking System built using Python.
This project was created to understand and practice Object-Oriented Programming (OOP) concepts like classes, objects, inheritance, and methods in a real-world scenario.

The system allows users to:

Park cars
Remove parked cars
View parking slot status
Calculate parking fees automatically
About the Project

Managing parking manually can become difficult when multiple vehicles are involved.
This project simulates a basic parking management system where parking slots are allocated automatically when a car enters the parking area.

The project is fully console-based and beginner-friendly, making it a great practice project for Python learners and freshers.

How the System Works
1. Park a Car

The user enters:

Vehicle Number
Owner Name
Parking Hours

The system automatically checks for an empty slot and parks the car.

2. Remove a Car

When a vehicle leaves:

The user enters the vehicle number
The parking fee is calculated
The slot becomes available again
3. Display Parking Status

Shows:

Occupied slots
Empty slots
Vehicle details
Parking Fee Calculation

Parking fee is calculated based on parking hours.

Fee = Parking Hours × ₹50

Example:

4 Hours = ₹200
